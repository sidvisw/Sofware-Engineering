This is a Map Managing System for Kharagpur.
It mainly gets driven by the unique ID given to nodes and ways in the map.osm xml file.

It has the following functions :
1. Get total number of nodes present in map.osm file.
2. Get the total number of ways present in map.osm file.
3. Search for a node element through its substring.
4. Find K-closest nodes to a given node.
5. Find the shortest path between two nodes.

You will require boost library to run this program-
1. For Linux - run the following command in the terminal
	$ sudo apt-get install libboost-all-dev
2. For Windows - visit the official boost website for instructions on how to install
	https://www.boost.org/doc/libs/1_74_0/more/getting_started/windows.html
